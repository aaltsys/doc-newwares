DOSWARES JSON Orders Exchange¶

DOSWARES ...

DOSWARES can send either Order or Shipment documents as responses when an order is shipped. An order document will contain all filled data elements including entries generated by the warehouse. The order response is documented first in this guide, as it presents the complete order data specification.

Notes¶

  1. Mandatory/Optional (M/O) indicates whether a data element is required or can be omitted in either the original customer order or the warehouse response.
  2. Code elements (referenceCode, dateCode, noteCode) are used to distinguish the meaning of following values. Currently code elements may be omitted because the following values are assigned to unique database entries.
