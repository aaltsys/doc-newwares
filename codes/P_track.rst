#############################
Tracking Codes
#############################

+-----------+----------------------------+
| Code      | Value                      |
+===========+============================+
|  P        | Track Product              |
+-----------+----------------------------+
|  C        | Track Lot Control          |
+-----------+----------------------------+
|  U        | Track Storage Units        |
+-----------+----------------------------+

.. note::
   These codes are internal to WARES operation and are not intended for data 
   exchange.
