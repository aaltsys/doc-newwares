#############################
Lot Control Codes
#############################

+-----------+----------------------------+
| Code      | Value                      |
+===========+============================+
| A3        | Control number             |
+-----------+----------------------------+
| B8        | Batch number               |
+-----------+----------------------------+
| HN        | Heat number                |
+-----------+----------------------------+
| JN        | Job number                 |
+-----------+----------------------------+
| KM        | Shelf life code            |
+-----------+----------------------------+
| LT        | Lot number                 |
+-----------+----------------------------+
| PJ        | Product date code          |
+-----------+----------------------------+ 
| PO        | Purchase order number      |
+-----------+----------------------------+
| RO        | Roll number                |
+-----------+----------------------------+
| SC        | Seller's date code         |
+-----------+----------------------------+
| SL        | Seller's lot number        |
+-----------+----------------------------+
| SN        | Serial number              |
+-----------+----------------------------+
| ZZ        | Mutually defined           |
+-----------+----------------------------+
+-----------+----------------------------+
| WD        | Warehouse date code        |
+-----------+----------------------------+
| WL        | Warehouse lot number       |
+-----------+----------------------------+
| NA        | Not assigned               |
+-----------+----------------------------+

Source: ASC-X12 Data Element 235 - Product/Service ID Qualifier

.. note::
   Codes `NA`, `WD`, and `WL` are not part of the ASC-X12 standards codes.
