#############################
Lot and Batch Control Codes
#############################

+-----------+----------------------------+
| Code      | Value                      |
+===========+============================+ 
| A3        | Control number             |
+-----------+----------------------------+ 
| B8        | Batch number               |
+-----------+----------------------------+ 
| HN        | Heat number                |
+-----------+----------------------------+ 
| JN        | Job number                 |
+-----------+----------------------------+ 
| LT        | Lot number                 |
+-----------+----------------------------+ 
| PJ        | Product date code          |
+-----------+----------------------------+ 
| PO        | Purchase order number      |
+-----------+----------------------------+ 
| RO        | Roll number                |
+-----------+----------------------------+ 
| SN        | Serial number              |
+-----------+----------------------------+ 

Source: ASC-X12 Data Element 235 - Product/Service ID Qualifier
